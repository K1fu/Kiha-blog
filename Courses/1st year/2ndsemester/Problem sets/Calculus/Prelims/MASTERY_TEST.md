## Question 1 - Complete Solution

**Part a)** $(f \circ g)(2)$

Step 1: Find $g(2)$
$$g(2) = 2^2 + 1 = 5$$

Step 2: Find $f(5)$
$$f(5) = 2(5) - 3 = 7$$

Therefore, $(f \circ g)(2) = 7$

**Part b)** $(g \circ f)(x)$

$$( g \circ f)(x) = g(f(x)) = g(2x-3)$$
$$= (2x-3)^2 + 1$$
$$= 4x^2 - 12x + 9 + 1$$
$$= 4x^2 - 12x + 10$$

**Part c)** Domain of $\frac{f}{g}(x)$

$$\frac{f}{g}(x) = \frac{2x-3}{x^2+1}$$

For domain, need $g(x) \neq 0$:
- $x^2 + 1 = 0$ has no real solutions
- Domain = $\mathbb{R}$ or $(-\infty, \infty)$

## Question 5 - Complete Solution

For continuity at $x = 1$, need:
$$\lim_{x \to 1^-} h(x) = h(1) = \lim_{x \to 1^+} h(x)$$

Step 1: Left-hand limit
$$\lim_{x \to 1^-} h(x) = \lim_{x \to 1^-} (x^2 - 1) = 0$$

Step 2: Function value
$$h(1) = 1^2 - 1 = 0$$

Step 3: Right-hand limit
$$\lim_{x \to 1^+} h(x) = \lim_{x \to 1^+} (2x + k) = 2 + k$$

Step 4: Set equal for continuity
$$2 + k = 0$$
$$k = -2$$

## Question 7 - Epsilon-Delta Proof

**Claim:** $\lim_{x \to 1} (3x + 2) = 5$

**Proof:**

Let $\epsilon > 0$ be given.

**Scratch Work:**
$$|(3x + 2) - 5| < \epsilon$$
$$|3x - 3| < \epsilon$$
$$3|x - 1| < \epsilon$$
$$|x - 1| < \frac{\epsilon}{3}$$

**Formal Proof:**

Choose $\delta = \frac{\epsilon}{3}$. Note that $\delta > 0$ since $\epsilon > 0$.

Assume $|x - 1| < \delta$.

Then:
$$|x - 1| < \frac{\epsilon}{3}$$
$$3|x - 1| < \epsilon$$
$$|3x - 3| < \epsilon$$
$$|(3x + 2) - 5| < \epsilon$$

Therefore, by the epsilon-delta definition, $\lim_{x \to 1} (3x + 2) = 5$. ∎

## Question 10 - Complete Solution

Evaluate $\lim_{h \to 0} \frac{(3+h)^2 - 9}{h}$

Step 1: Expand numerator
$$(3+h)^2 - 9 = 9 + 6h + h^2 - 9 = 6h + h^2$$

Step 2: Factor
$$\frac{6h + h^2}{h} = \frac{h(6 + h)}{h}$$

Step 3: Cancel h (valid since $h \neq 0$ in limit)
$$= 6 + h$$

Step 4: Evaluate limit
$$\lim_{h \to 0} (6 + h) = 6$$

**Interpretation:** This is the derivative of $f(x) = x^2$ at $x = 3$.

## Question 13 - Complete Solutions

**Part a)** $\lim_{x \to 1} \frac{x^3 - 1}{x - 1}$

Step 1: Factor numerator (difference of cubes)
$$x^3 - 1 = (x-1)(x^2 + x + 1)$$

Step 2: Cancel common factor
$$\frac{(x-1)(x^2 + x + 1)}{x-1} = x^2 + x + 1$$

Step 3: Evaluate limit
$$\lim_{x \to 1} (x^2 + x + 1) = 1 + 1 + 1 = 3$$

**Part b)** $\lim_{x \to 4} \frac{\sqrt{x} - 2}{x - 4}$

Step 1: Multiply by conjugate
$$\frac{\sqrt{x} - 2}{x - 4} \cdot \frac{\sqrt{x} + 2}{\sqrt{x} + 2} = \frac{x - 4}{(x-4)(\sqrt{x} + 2)}$$

Step 2: Cancel common factor
$$= \frac{1}{\sqrt{x} + 2}$$

Step 3: Evaluate limit
$$\lim_{x \to 4} \frac{1}{\sqrt{x} + 2} = \frac{1}{2 + 2} = \frac{1}{4}$$

## Question 18 - Epsilon-Delta Proof

**Claim:** $\lim_{x \to 2} x^2 = 4$

**Proof:**

Let $\epsilon > 0$ be given.

**Scratch Work:**
$$|x^2 - 4| < \epsilon$$
$$|(x-2)(x+2)| < \epsilon$$
$$|x-2| \cdot |x+2| < \epsilon$$

We need to bound $|x+2|$. Assume $|x-2| < 1$, then $1 < x < 3$, so $3 < x+2 < 5$.
Thus $|x+2| < 5$.

Then: $|x-2| \cdot 5 < \epsilon$, so $|x-2| < \frac{\epsilon}{5}$

**Formal Proof:**

Choose $\delta = \min\{1, \frac{\epsilon}{5}\}$. Note that $\delta > 0$.

Assume $|x - 2| < \delta$.

Since $\delta \leq 1$, we have $|x - 2| < 1$, which gives $1 < x < 3$.
Therefore, $3 < x + 2 < 5$, so $|x + 2| < 5$.

Also, since $\delta \leq \frac{\epsilon}{5}$, we have $|x - 2| < \frac{\epsilon}{5}$.

Then:
$$|x^2 - 4| = |x-2| \cdot |x+2| < \frac{\epsilon}{5} \cdot 5 = \epsilon$$

Therefore, by the epsilon-delta definition, $\lim_{x \to 2} x^2 = 4$. ∎

## Question 22 - Complete Solution

For f to be continuous everywhere, it must be continuous at x = 1 and x = 2.

**At x = 1:**
$$\lim_{x \to 1^-} f(x) = 3$$
$$f(1) = 3$$
$$\lim_{x \to 1^+} f(x) = a(1) + b = a + b$$

For continuity: $a + b = 3$ ... (Equation 1)

**At x = 2:**
$$\lim_{x \to 2^-} f(x) = a(2) + b = 2a + b$$
$$f(2) = 5$$
$$\lim_{x \to 2^+} f(x) = 5$$

For continuity: $2a + b = 5$ ... (Equation 2)

**Solving the system:**
From Equation 2 - Equation 1:
$$(2a + b) - (a + b) = 5 - 3$$
$$a = 2$$

Substituting into Equation 1:
$$2 + b = 3$$
$$b = 1$$

Therefore, $a = 2$ and $b = 1$.

**Verification:**
- At $x = 1$: all limits and function value equal 3 ✓
- At $x = 2$: all limits and function value equal 5 ✓
