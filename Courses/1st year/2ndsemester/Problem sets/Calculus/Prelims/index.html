Key Concepts
Definition: Relation

A relation from set A to set B is any nonempty subset of A √ó B.

Definition: Function

A function from A to B is a relation where each element a ‚àà A corresponds to exactly one element b ‚àà B.

$$f: A \to B$$

üîë Key Difference: In a function, no two ordered pairs have the same first component.
Example 1:

Which of the following are functions?

a) \( \{(1,2), (3,4), (1,5)\} \) ‚Äî NOT a function (1 maps to both 2 and 5)

b) \( \{(1,2), (3,2), (5,2)\} \) ‚Äî IS a function (each input maps to exactly one output)

Vertical Line Test

A graph represents a function if and only if no vertical line intersects the graph more than once.

Quick Reference Formula Sheet

Limit Laws

$$\lim_{x \to b} c = c$$

$$\lim_{x \to b} x = b$$

$$\lim_{x \to b} [f(x) \pm g(x)] = \lim_{x \to b} f(x) \pm \lim_{x \to b} g(x)$$

$$\lim_{x \to b} [f(x) \cdot g(x)] = \lim_{x \to b} f(x) \cdot \lim_{x \to b} g(x)$$

$$\lim_{x \to b} \frac{f(x)}{g(x)} = \frac{\lim_{x \to b} f(x)}{\lim_{x \to b} g(x)}$$ if $\lim_{x \to b} g(x) \neq 0$

Epsilon-Delta Definition

$$\lim_{x \to b} f(x) = L \iff \forall \epsilon > 0, \exists \delta > 0: 0 < |x - b| < \delta \implies |f(x) - L| < \epsilon$$

Continuity

f is continuous at b if and only if:

  1. f(b) exists
  2. $\lim_{x \to b} f(x)$ exists
  3. $\lim_{x \to b} f(x) = f(b)$
