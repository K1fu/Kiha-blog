1.1 Relations & Functions

Learning Objectives
  • Distinguish between relations and functions
  • Apply the vertical line test
  • Use function notation correctly
Definition: Function

A function from A to B is a relation where each element a ‚àà A corresponds to exactly one element b ‚àà B.

Vertical Line Test

A graph represents a function if and only if no vertical line intersects the graph more than once.

Operations ‚Ä¢ 40 min

1.2 Function Operations

Composition

$$(f \circ g)(x) = f(g(x))$$

‚ö†Ô∏è Warning

Composition is NOT commutative: $$(f \circ g)(x) \neq (g \circ f)(x)$$ in general.

Domain & Range ‚Ä¢ 35 min

1.3 Domain and Range

Domain $$D_f$$

The set of all permissible input values (x-values) for which the function is defined.

Example

For $$f(x) = \frac{1}{x-3}$$, we need $$x-3 \neq 0$$, so $$x \neq 3$$.

Domain: $$(-\infty, 3) \cup (3, \infty)$$

1.5 Inverse Functions

Horizontal Line Test

A function is one-to-one (and has an inverse) if and only if no horizontal line intersects its graph more than once.

‚àû Module 2: Limits

7 Lessons ‚Ä¢ 135 Problems
Introduction ‚Ä¢ 45 min

2.1 Introduction to Limits

Intuitive Definition

We write $$\lim_{x \to b} f(x) = L$$ if f(x) gets arbitrarily close to L as x approaches b.

Rigorous Proofs ‚Ä¢ 90 min

2.2 Epsilon-Delta Definition

Formal Definition

$$\lim_{x \to b} f(x) = L$$ means:

For all $$\epsilon > 0$$, there exists $$\delta > 0$$ such that:

$$0 < |x - b| < \delta \implies |f(x) - L| < \epsilon$$

Basic Theorems ‚Ä¢ 40 min

2.3 Basic Limit Laws

$$\lim_{x \to b} [cf(x)] = c \cdot \lim_{x \to b} f(x)$$ $$\lim_{x \to b} [f(x) \pm g(x)] = L \pm M$$
Algebraic Methods ‚Ä¢ 50 min

2.4 Algebraic Limit Theorems

Quotient Rule

$$\lim_{x \to b} \frac{f(x)}{g(x)} = \frac{L}{M}$$ provided $$M \neq 0$$. If $$M=0$$, further analysis is required.

Directional Limits ‚Ä¢ 45 min

2.5 One-Sided Limits

Existence Theorem

The limit exists if and only if both left and right limits exist and are equal.

Asymptotes ‚Ä¢ 50 min

2.6 Infinite Limits

Vertical Asymptote

Line $$x=a$$ is a VA if $$\lim_{x \to a^\pm} f(x) = \pm\infty$$.

End Behavior ‚Ä¢ 45 min

2.7 Limits at Infinity

Degree Rule

For rational functions, compare degrees of numerator and denominator to determine the horizontal asymptote.

3.1 Continuity at a Point

Three Conditions
  1. $$f(c)$$ is defined
  2. $$\lim_{x \to c} f(x)$$ exists
  3. $$\lim_{x \to c} f(x) = f(c)$$
Theorems ‚Ä¢ 45 min

3.2 Continuity Theorems

Intermediate Value Theorem

If f is continuous on [a,b] and k is between f(a) and f(b), there exists c in (a,b) such that f(c) = k.

  • Infinite (Asymptote)

Formula Reference

Limit Laws

$$\lim_{x \to c} [f(x) \pm g(x)] = \lim_{x \to c} f(x) \pm \lim_{x \to c} g(x)$$ $$\lim_{x \to c} [f(x) \cdot g(x)] = \lim_{x \to c} f(x) \cdot \lim_{x \to c} g(x)$$ $$\lim_{x \to c} \frac{f(x)}{g(x)} = \frac{\lim_{x \to c} f(x)}{\lim_{x \to c} g(x)}$$

Epsilon-Delta Definition

$$\lim_{x \to c} f(x) = L \iff \forall \epsilon > 0, \exists \delta > 0 : 0 < |x - c| < \delta \implies |f(x) - L| < \epsilon$$

Continuity Definition

$$f \text{ is continuous at } c \iff \lim_{x \to c} f(x) = f(c)$$
